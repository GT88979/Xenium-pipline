## 四、函数索引（按环境分类）

---

### 环境一：ViaEnv

#### 4.1 数据加载

**`load_xenium_data(data_dir)`**
从 Xenium 原始输出（h5 + csv）构建 AnnData。

```python
adata = xp.load_xenium_data('/path/to/Xenium_output/')
```

---

#### 4.2 质控与预处理

**`basic_qc_filtering(adata, min_reads=10, min_genes=5)`**
Nature Methods 2025 标准质控过滤。

```python
adata = xp.basic_qc_filtering(adata, min_reads=10, min_genes=5)
xp.plot_qc_metrics(adata)
```

**`preprocess_xenium(adata, target_sum=100, use_log=True, use_scale=True, n_pcs=None)`**
标准化预处理：Library Norm → Log1p → Scale → PCA。

```python
adata = xp.preprocess_xenium(adata, target_sum=100, use_log=True, use_scale=True)
```

---

#### 4.3 聚类

**`build_knn_and_cluster(adata, n_neighbors=16, resolutions=[0.1,0.3,0.6], use_rep='X_pca')`**
Leiden 聚类（多分辨率）。

```python
adata = xp.build_knn_and_cluster(adata, n_neighbors=16, resolutions=[0.1, 0.3, 0.6])
```

**`RunSpatialcluster_Banksy(adata, spatial_key='spatial', lambda_val=0.2, res_list=[0.1,0.2,0.6])`**
BANKSY 空间聚类（需预先运行 `sc.pp.highly_variable_genes` 选 HVG）。

```python
sc.pp.highly_variable_genes(adata, flavor='seurat', n_top_genes=2000)
adata = xp.RunSpatialcluster_Banksy(adata, lambda_val=0.2, res_list=[0.1, 0.2, 0.6])
```

> ⚠️ 多样本推荐用 cellcharter 环境的 `run_cellcharter_pipeline`，更稳健。

---

#### 4.4 下游空间分析

**`plot_spatial_overview(adata, cluster_key, highlight_types, grid_interval=1000)`**
空间分布总览图，返回各细胞类型坐标范围。

```python
ranges = xp.plot_spatial_overview(
    adata,
    cluster_key='Cell_type',
    highlight_types=['Treg', 'Carcinoma'],
    grid_interval=1000,
)
# ranges: {'Treg': (x_min, x_max, y_min, y_max), ...}
```

**`extract_roi(adata, x_range, y_range)`**
按坐标范围提取 ROI 子区域。

```python
roi_data = xp.extract_roi(adata, x_range=(3200, 4200), y_range=(19455, 21455))
```

**`calculate_shortest_distance(adata, target_subset_cell, niche_cells, cluster_key)`**
核心公式 D_k = min_j d(c_k, c_j)，KDTree 加速。

```python
dists = xp.calculate_shortest_distance(adata, 'Carcinoma', ['Treg', 'T'], 'Cell_type')
```

**`plot_spatial_distance_kde(adata, center_types, target_types, ...)`**
最短距离密度分布图（KDE），返回统计量 DataFrame。

```python
stats = xp.plot_spatial_distance_kde(
    roi_data,
    center_types=['Treg'],
    target_types=['Inflammatory CAF', 'B'],
    max_dist=500,
)
print(stats)
```

**`run_spatial_enrichment_pipeline(adata, center_cell, radius=50)`**
空间邻域富集分析（半径内外卡方检验）。

```python
res = xp.run_spatial_enrichment_pipeline(adata, center_cell='Carcinoma', radius=50)
```

**`plot_niche_composition_donuts(adata, cluster_key, cell_type_key, threshold=0.03)`**
每个空间域的细胞组成环形图。

```python
xp.plot_niche_composition_donuts(
    adata,
    cluster_key='cluster_cellcharter',
    cell_type_key='Cell_type',
    threshold=0.03,
)
```

### 环境二：cellcharter

> ⚠️ 必须切换到 cellcharter 环境重启 kernel 后使用。

---

#### 4.5 空间聚类

**`run_cellcharter_pipeline(adata, batch_key='patient', layer='counts', n_layers=3, min_k=2, max_k=10)`**
CellCharter 自动 K 值空间聚类（scVI 批次校正 + AutoK 稳定性搜索）。

```python
adata = xp.run_cellcharter_pipeline(
    adata,
    batch_key='patient',
    layer='counts',
    n_layers=3,
    min_k=2,
    max_k=10,
)
# 结果写入 adata.obs['cluster_cellcharter']
```

**返回**：adata（inplace 更新 `obs['cluster_cellcharter']`、`obsm['X_scVI']`、`obsm['X_cellcharter']`）

---

#### 4.6 配受体分析

**`run_spatial_lr_pipeline(adata, cluster_key='Cell_type', n_perms=1000, n_jobs=8)`**
空间配受体置换检验，返回 sq.gr.ligrec() 原生结果。

```python
res = xp.run_spatial_lr_pipeline(
    adata,
    cluster_key='Cell_type',
    n_perms=1000,
    n_jobs=8,     # 不要用 -1，核数过高会导致 AssertionError
)
```

**返回**：dict（键：`means`、`pvalues`），直接传给 `sq.pl.ligrec()` 做可视化。

```python
# 全部热图
sq.pl.ligrec(res, cluster_key='Cell_type', pvalue_threshold=0.05)

# 聚焦特定互作
sq.pl.ligrec(res, cluster_key='Cell_type',
             source_groups=['Treg'],
             target_groups=['Carcinoma', 'TAM1'],
             means_range=(0.3, np.inf),
             annotate=True)
```

## 七、常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| `AssertionError: Expected n_jobs results` | `n_jobs=-1` 核数过高任务调度失败 | 改用 `n_jobs=8` 或 `n_jobs=24` |
| co_occurrence 极慢 | 175k 细胞 + interval=50 计算量爆炸 | 减小 `interval=10`，加 `n_splits=4` |
| co_occurrence 图挤在一行 | squidpy 内置限制 | 用 `clusters=['Treg']` 一次画一个 |
| 换环境后函数找不到 | ViaEnv 函数不在 cellcharter 环境 | 重启 kernel，重新 `import xenium_pipline as xp` |
| n_jobs=-1 在 LR 分析报错 | 核数解析为 384，实际任务返回 334 个 | 改用 `n_jobs=8` 或 `n_jobs=24` |
